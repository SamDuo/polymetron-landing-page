Polymetron — AI-Native Spatial Intelligence for Real Estate & Infrastructure
============================================================================

Static site (HTML + CSS + JS). No build step required.

SETUP:
1. Get a free Mapbox token at https://account.mapbox.com/
2. Open main.js and replace the MAPBOX_TOKEN value on line 9
3. Deploy to any static host (Netlify, Vercel, GitHub Pages, etc.)

FORM:
Replace REPLACE_WITH_YOUR_FORM_ID in index.html with your Formspree ID.

FILES:
  index.html   — Main page
  styles.css   — Legora-inspired design system
  main.js      — Mapbox maps, AI chat, scoring, animations
  favicon.png  — Site icon
